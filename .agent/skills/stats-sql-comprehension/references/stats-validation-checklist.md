# 統計解析＆RWDパイプライン妥当性チェックリスト

## 1. データ抽出・コホート定義のチェック

- [ ] **選択バイアス (Selection Bias)**: 特定の条件でサンプルを除外した結果、一般化可能性が損なわれていないか。
- [ ] **不死時間バイアス (Immortal Time Bias)**: イベント発生前の期間が誤ってフォローアップ期間に含まれていないか。
- [ ] **生存者バイアス (Survivor Bias)**: ある時点で生存していた患者のみを集計対象にしていないか。

## 2. 変数・前処理のチェック

- [ ] **欠損値ハンドリング**: 欠損メカニズム（MCAR, MAR, MNAR）の考慮と代入法の妥当性。
- [ ] **外れ値処理**: 単なるカットオフによる情報損失のリスク評価。

## 3. 統計推論・検定のチェック

- [ ] **前提条件の検証**: 検定手法（t検定、ANOVA、回帰分析等）の前提条件（正規性、等分散性、線形性）が満たされているか。
- [ ] **交絡因子の調整**: 傾向スコア（Propensity Score）、多変量調整が適切に行われているか。
