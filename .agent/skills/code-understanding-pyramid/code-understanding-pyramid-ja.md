# code-understanding-pyramid 日本語案内

このファイルは日本語利用者向けの案内です。実行契約の正本は同じディレクトリの `SKILL.md` です。

本Skillは単独で成果物を保存するのではなく、`code-understanding-pro` の下位フレームとして次の順序を提供します。

1. 文脈把握
2. 構造・全体像
3. ロジックとデータの追跡
4. 設計意図、テスト、リスク
5. ドキュメント、レビュー、リファクタリングへの活用

Quick Mode以外の結果は、親Skillの共通契約に従って `report.md` へ統合します。出力先、必須節、品質検証は `../code-understanding-pro/references/interface.md` を参照してください。
