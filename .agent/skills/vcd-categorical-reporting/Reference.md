# リファレンス: vcd-categorical-reporting

AIによる分析考察の自動生成と、品質評価基準に関するリファレンスです。

## 1. 考察生成の論理構成

本スキルでは、以下の優先順位で統計事実を言語化します。

1.  **実用的意義（効果量）**: Cramér's V が「実務的に意味のある大きさ」か。
2.  **局所的特異性（残差）**: どのセグメント（セル）が期待値から乖離しているか。
3.  **統計的確信度（ベイズ因子・P値）**: その偏りは偶然ではないと言えるか。

## 3. レポート品質の評価
- **正確性**: 統計数値を誤解なく引用しているか。
- **有用性**: 単なる数値の羅列ではなく、次のアクション（施策変更、追加調査等）への示唆を含んでいるか。
