# AI判断基準

## 1. 残差の閾値評価

Pearson残差の絶対値に基づき、セルごとの偏りの強さを以下の基準で言語化すること。

| 閾値（絶対値） | 有意水準相当 | 言語化の目安 |
| :--- | :--- | :--- |
| `≥ 1.96` | 5% | 「偏りが見られる」「傾向がある」 |
| `≥ 2.58` | 1% | 「顕著な偏り」「強い傾向」 |
| `≥ 3.29` | 0.1% | 「極めて特異な偏り」「特筆すべき集中・忌避」 |
| `< 1.96` | 有意ではない | 言及しない（ランダムな誤差の範囲） |

## 2. 第2段階の交互作用評価（2-way モデルの残差）

2-way モデルの残差は「AとBの単純な相関だけでは説明がつかない、第3の変数Cによる特異な組み合わせ」を示します。
これを言語化する際は「〜だから〜という単純な理由ではなく、〜の場合にのみ〜が選ばれやすい」といった洞察を含めること。

## 3. 層別選択ロジック（`strata_summary`）

`max_abs_res_per_stratum`（最大残差）と `cramers_v_per_stratum`（Cramer's V）から、**レポートの第2章に前面配置すべき1つの層**を以下の優先度で選択すること。

1. **極端な外れ値を持つ層**: `max_abs_res_per_stratum` が 3.29 以上である層。
2. **全体的な関連が強い層**: `cramers_v_per_stratum` が他の層と比べて顕著に高い層（> 0.3 など）。
3. **ビジネス的関心**: 上記が同等の場合は、一般名詞として特異なもの（「その他」よりも「痛くなりそうなとき」など）を優先。

## 4. 有意セル比率の解釈

`n_significant_cells_5pct` が `total_cells` の 5% を大きく上回る（例：30%のセルが有意など）場合、特定のセルだけが外れ値なのではなく、モデル全体がデータ構造を捉えきれていない（変数間に強固な構造的関連がある）ことを意味します。
この場合、レポートの `> [!NOTE]` に「特定のセルの偏りというより、全体として強い構造的関連が存在します」と注釈を付与すること。

## 5. 文体とスタイル

- 断定的な表現を避け、「〜の傾向が示唆される」「〜という偏りが観察された」といった客観的な記述に努めること。
- 推奨アクションは、ビジネスエグゼクティブ（非統計専門家）が理解でき、次の意思決定に繋がる具体的な提案（1〜2文）とすること。
