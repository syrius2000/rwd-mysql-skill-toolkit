# テーブル名 → 意味（例）

DB の TABLE_COMMENT が無い場合や、日本語ラベルを統一したい場合に参照する。スキル実行時にユーザーが口頭で渡してもよい。未記載のテーブルは英名のみで ER 図に含めてよい。

| テーブル名（英） | 意味（日本語） |
|------------------|----------------|
| inpatient        | 入院           |
| outpatient       | 外来           |
| orders           | オーダー       |
| patient          | 患者           |
| prescription     | 処方           |
| diagnosis        | 診断           |
| physician        | 医師           |
| department       | 診療科         |
