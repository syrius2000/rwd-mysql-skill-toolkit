# VCD 分析 AI 判断基準

## 残差の解釈基準

| 閾値 | 統計的意味 | レポートでの扱い |
| :--- | :--- | :--- |
| \|res\| ≥ 1.96 | 5%水準で有意 | 第2章の有意セル表に含める |
| \|res\| ≥ 2.58 | 1%水準で有意 | 第1章の所見で言及必須 |
| \|res\| ≥ 3.29 | 0.1%水準で有意 | 最も注目すべき発見として強調 |

## 残差の方向性

- **正の残差（青）**: 期待値より**多い**。その組み合わせに正の関連がある。
- **負の残差（赤）**: 期待値より**少ない**。その組み合わせに負の関連（回避・排反）がある。

## 層別選択ロジック（3-way）

AI が `strata_summary` から第2章に前面配置する層を選ぶ基準：

1. **max_abs_res_per_stratum** が最大の層 → 最も顕著な偏りの存在を示す
2. **cramers_v_per_stratum** が最大の層 → 全体的な関連性が最も強い
3. 上記2つが異なる場合 → 両方を前面配置し、理由を比較記述

### 有意セル比率の判断

| n_significant_cells_5pct / total_cells | 解釈 | 注釈 |
| :--- | :--- | :--- |
| < 5% | 偶然レベル | 「有意な偏りは認められない」 |
| 5-20% | 部分的な関連 | 特定セルの偏りに注目 |
| > 20% | 広範な関連 | 「変数間に体系的な関連が存在する」 |

## 記述スタイル（ハイブリッド）

- **サマリー文**: 結論を1-2文で要約。「〜が判明した」ではなく「〜である」と断定する。
- **箇条書き所見**: 残差値と具体的な解釈をセットで。数値は必ず記載。
- **推奨アクション**: ビジネス上の次のステップを具体的に（「追加分析を推奨」ではなく「〜の層別分析が有効」）。

## 判断の2段階思考

### 第1段階：全体構造の俯瞰（主効果モデル）

主効果モデル（`model_type == "Main"`）の残差から：
- 変数間の「自明かつ強力な関連性」を指摘する
- 期待値からの乖離が最も大きいセルに注目
- 全体的なパターン（例：「カテゴリAの水準1は全体的に高い」）を記述

### 第2段階：局所交互作用の洞察（2-way モデル）

2-way モデル（`model_type == "2-Way"`）の残差から：
- 主効果で説明できない「特異な組み合わせの偏り」を言語化
- シンプソンのパラドックスや潜在的交互作用の可能性に言及
- 「主効果モデルでは [セル] が |res| = X だったが、2-way では |res| = Y に低下した → このセルの偏りは交互作用で説明される」のような比較記述を活用
