# 高度な分析手法と実用的ワークフロー

このドキュメントは、大規模データにおける「実用的有意性」の抽出と、複雑な関連性を解明するための高度な分析パターンをまとめたものです。

## 1. Dual-Filter フレームワーク

統計的有意性（P値）の飽和を回避し、真に価値のあるインサイトのみを抽出するための 2 段階フィルタリング。

1. **Step 1: 全体スクリーニング (Effect Size Filter)**
    - Cramér's V または Fei ($\text{פ}$) を算出し、あらかじめ設定した閾値（例: $> 0.1$）を下回る場合は「統計的に有意でも実質的な意味はない」と判断して分析を打ち切る、あるいは警告を出す。
2. **Step 2: セル単位の深掘り (Evidence Filter)**
    - スクリーニングを通過したデータに対してのみ、エビデンススコア ($r^2 - k\log N$) やベイズ因子を用いて、どの組み合わせが特異的かを特定する。

---

## 2. 高度な関連性抽出手法

### アソシエーション分析 (ARM)
リフト値（Lift）を用いて、サンプルサイズに依存しない関連の強さを評価します。
$$\text{Lift}(A \to B) = \frac{P(B|A)}{P(B)}$$
- **$Lift > 1$**: A が起こるとき、B も起こりやすい（正の相関）。
- **$Lift \approx 1$**: A と B は独立。

### スパース・コレスポンデンス分析 (Sparse CA)
L1 正則化を適用し、関連性の弱いカテゴリをゼロに縮退させることで、可視化の際のノイズを除去し、主要な関連性のみを強調します。

---

## 3. 実践的改善パターン

- **Top-K Ranking**: 全セルの列挙を廃止し、エビデンススコアの上位 5〜20 個に限定して提示する。
- **動的視覚化**: モザイクプロット等の配色閾値を、残差の絶対値ではなく、パーセンタイルやエビデンススコアに基づいて動的に変更する。
- **大規模データモードの自動判定**: $N > 5000$ 程度の規模で、自動的に効果量重視の解釈（ナラティブ生成）へ切り替える。
